# Дашборд аналитики продаж в Power BI

## О проекте
Аналитический отчёт в Power BI о продажах интернет-магазина.
**Цель дашборда:**
Помочь руководителю и менеджерам видеть ключевые метрики, отслеживать динамику, находить убыточные заказы и анализировать продажи.


## Что реализовано
- Загрузка и очистка данных в Power Query
- Создание календаря
- 6 ключевых метрик (Выручка, Прибыль, Рентабельность, АКБ, Средний чек, Затраты на логистику)
- Динамика показателей по месяцам
- Анализ по сегментам покупателей
- Drill-down по категориям товаров
- Карта регионов
- Таблица менеджеров с убытками
